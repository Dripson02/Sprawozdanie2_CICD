# Bezpieczeństwo_Procesów_CI/CD
- Skład grupy: Katarzyna Jakubowska, Jakub Iskra
- Temat: System zarządzania zadaniami
- Opis: Aplikacja internetowa “to do list”, do wyświetlania zadań, dodawania nowych, usuwania istniejących oraz zmiany ich statusu. System oparty na 3 mikrousługach: frontend do interfejsu użytkownika, backend do przetwarzania danych oraz logiki biznesowej, baza danych do przechowywania danych.
